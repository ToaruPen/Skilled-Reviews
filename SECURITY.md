# Security Policy

## Supported Versions

This project is currently maintained on a best-effort basis.

- Supported: the latest `main` and the latest tagged release
- Not supported: older tags

## Reporting a Vulnerability

Please report security issues **privately**.

Preferred:
- GitHub Security Advisories (private report): https://github.com/ToaruPen/Skilled-Reviews/security/advisories/new

If you cannot use GitHub Security Advisories:
- Open a GitHub issue with **minimal details** and ask to move to a private channel.

What to include:
- A short description and impact
- Reproduction steps / PoC (if available)
- Affected versions (tag/commit) and environment
- Any suggested fix or mitigation

# セキュリティポリシー（日本語）

## サポート対象バージョン

本プロジェクトは現時点ではベストエフォートで運用しています。

- サポート対象: 最新の `main` と最新のタグ付きリリース
- サポート対象外: それ以前のタグ

## 脆弱性の報告方法

セキュリティ上の問題は **公開せず** にご連絡ください。

推奨:
- GitHub Security Advisories（非公開レポート）: https://github.com/ToaruPen/Skilled-Reviews/security/advisories/new

GitHub Security Advisories が使えない場合:
- GitHub Issue を作成し、**詳細は最小限** にして「非公開で共有する手段」を相談してください。

含めてほしい情報:
- 概要と影響範囲
- 再現手順 / PoC（可能なら）
- 影響バージョン（tag/commit）と環境情報
- 既に考えている回避策や修正案（任意）
